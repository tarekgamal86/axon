# Axon - EPCM Project Management Platform

Next-generation agentic flow end-to-end project management, design and construction management tool for the EPCM (Engineering, Procurement, Construction, Management) industry.

## Overview

Axon addresses the pain points of managing €6M-€70M CAPEX projects in the engineering and construction industry through intelligent automation, real-time collaboration, and AI-assisted workflows.

## Target Users

- **Project Owners** - Track progress, manage budgets, ensure compliance
- **Engineering Firms** - Design coordination, document management, change orders
- **Contractors** - Field execution, resource management, quality control
- **Suppliers** - Delivery tracking, invoice management, compliance docs
